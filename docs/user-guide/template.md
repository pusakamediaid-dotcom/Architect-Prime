# Panduan Template Akademik

Menu **Template** menyediakan format siap salin untuk:

- rencana belajar mingguan;
- outline makalah;
- checklist presentasi;
- catatan Cornell;
- rencana proyek kelompok;
- target semester;
- evaluasi nilai.

Klik **Salin Template**, lalu tempel ke aplikasi catatan Anda.
