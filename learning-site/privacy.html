Privasi

Data Anda tersimpan di browser

Pusaka Student Hub versi static/local-first tidak mengirim tugas, jadwal, nilai, checklist, target, atau catatan Anda ke server.

  • Data disimpan di browser perangkat menggunakan localStorage.
  • Anda bisa export data sebagai JSON untuk backup.
  • Anda bisa import data JSON kembali ke browser.
  • Anda bisa reset semua data kapan saja.
  • Jika fitur cloud sync ditambahkan di masa depan, kebijakan privasi akan diperbarui.
