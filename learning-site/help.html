Bantuan

Cara memakai Pusaka Student Hub

Tugas

Isi nama tugas, mata kuliah, deadline, prioritas, dan status. Anda bisa menandai selesai atau menghapus tugas.

Jadwal

Tambahkan hari, jam, aktivitas, dan catatan lokasi.

Nilai

Tambahkan komponen nilai beserta bobot dan skor. Total bobot tidak boleh lebih dari 100%.

Checklist

Buat checklist sendiri atau gunakan template presentasi, ujian, makalah, dan skripsi/proyek akhir.

Export/Import

Gunakan export JSON untuk backup. Import JSON untuk memulihkan data di browser lain.
