# Architect Prime Learning Hub

Website edukasi gratis yang mengubah materi teknis Architect-Prime menjadi portal belajar responsif untuk mahasiswa, pelajar, dan pemula.

## Menjalankan secara lokal

```bash
cd website
npm install
npm run dev
```

## Build produksi

```bash
cd website
npm install
npm run build
```

## Pengaturan Vercel

- Framework Preset: Vite
- Root Directory: `website`
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

Versi awal bersifat statis, tidak membutuhkan API key, basis data, atau layanan berbayar.
