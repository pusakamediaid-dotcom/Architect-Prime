# Support

For issues, open a GitHub issue with:

1. Module name.
2. Operating system.
3. Runtime versions.
4. Exact command executed.
5. Full error output.

Commercial users should maintain a private fork and pin dependency versions before production deployment.

## Pusaka Student Hub Support Notes

Untuk pengguna awam:

1. Buka website Vercel.
2. Isi data tugas, jadwal, nilai, checklist, target, atau catatan.
3. Gunakan Export JSON untuk backup.
4. Jika data hilang setelah membersihkan cache browser, pulihkan dari file export jika tersedia.

Untuk laporan bug, sertakan perangkat, browser, langkah yang dilakukan, dan screenshot jika memungkinkan.
