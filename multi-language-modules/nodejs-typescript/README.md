# Node.js TypeScript Module

Production-ready Node.js boilerplate with TypeScript, Express, and best practices.

## Structure

```
nodejs-typescript/
├── src/
│   ├── controllers/
│   ├── services/
│   ├── repositories/
│   ├── models/
│   ├── middleware/
│   ├── routes/
│   ├── utils/
│   └── config/
├── tests/
├── docker/
└── package.json
```

## Features

- Express.js REST API
- TypeScript strict mode
- Prisma ORM
- JWT Authentication
- Request validation
- Docker containerization
