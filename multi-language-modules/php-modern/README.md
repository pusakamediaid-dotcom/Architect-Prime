# PHP Modern Module

Modern PHP boilerplate with Laravel clean architecture patterns.

## Structure

```
php-modern/
├── app/
│   ├── Http/
│   │   ├── Controllers/
│   │   ├── Middleware/
│   │   └── Requests/
│   ├── Services/
│   ├── Repositories/
│   ├── Models/
│   └── Providers/
├── config/
├── database/
│   ├── migrations/
│   └── seeders/
├── routes/
├── tests/
└── docker/
```

## Features

- Laravel Service Layer pattern
- Repository pattern
- Form Request validation
- Clean controllers
- Dependency injection
